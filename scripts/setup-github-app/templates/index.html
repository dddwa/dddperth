GitHub App Setup

Create a GitHub App for DDD Perth admin authentication.

This tool drives GitHub's manifest flow and writes the resulting OAuth credentials for one environment.

The Worker only needs WEBSITE_GITHUB_APP_CLIENT_ID and WEBSITE_GITHUB_APP_CLIENT_SECRET.

Must be globally unique on GitHub.
Used as the OAuth callback base — for local this is http://localhost:3800.
For production, use dddwa.

For staging/production, the Worker must already exist (run wrangler deploy --env <env> once first), and your shell needs CLOUDFLARE_API_TOKEN + CLOUDFLARE_ACCOUNT_ID set.
